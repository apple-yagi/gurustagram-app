# Gurustagram

## 概要
自分の行った飲食店の情報を投稿できるWebアプリケーションです。

お店の情報とともに、コメントも一緒に投稿することができます。

URL：https://gurustagram-app.firebaseapp.com/#/

## 機能
### ユーザー機能
 ・作成
 
 ・編集（名前、プロフィール画像）
 
 ・ログイン
 
### 投稿記事
 ・表示
 
 ・作成
 
 ・検索
